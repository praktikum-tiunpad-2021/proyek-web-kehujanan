## Fungsi

- Login sebagai user<br>
- Mencatat tugas yang diberikan beserta mata kuliah dan deadlinenya<br>
- Menambah tag-tag pada tugas<br>
- Ceklis status tugas yang sudah selesai<br>
- Filter tugas<br>
- Mengedit tugas<br>
- Login dengan akun lain<br>
